# LSTM-Stock-Price-Prediction
A python application which predicts and visualises ASX stock market prices using tensorflow LSTM models. A jupyter notebook version is included as well.

# Dependencies
>pip install numpy
>
>pip install pandas
>
>pip install matplotlib
>
>pip install tensorflow
>
>pip install sklearn

# Run
>Create a directory called Data in the project directory.
>
>Find the ASX code of your company.
>
>Go to Yahoo Finance and download the time series data for your company into the Data directory.
>
>Run python main.py in terminal.
